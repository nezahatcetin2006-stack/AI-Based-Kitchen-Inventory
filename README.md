# AI Based Kitchen Inventory

Evdeki gıdaların envanterini tutup **son kullanıma yaklaşan ürünleri** görsel olarak uyaran, ayrıca ileride AI ile fotoğraftan tanıma ve tarif önerme hedefleyen uygulama.

## Stack (önerilen)
- Frontend: Next.js (React, TypeScript, Tailwind)
- Backend: FastAPI (Python)
- Database: PostgreSQL (local geliştirme için `docker-compose.yml`)

## Hızlı başlangıç
1. Gerekli runtime’ları kurun: **Node.js**, **Python 3.10+**, **Docker** (opsiyonel ama DB için önerilir).
2. Backend `.env.example` dosyasına göre ayarları yapın.
3. Backend’i başlatın.
4. Frontend’i başlatın.
